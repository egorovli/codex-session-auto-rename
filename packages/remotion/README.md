# Codex Session Auto Rename Remotion showcase

Short Remotion showcase for the Codex Session Auto Rename plugin.

## Commands

```sh
npm run studio
npm run compositions
npm run render:still
npm run render
npm run typecheck
```

The main video renders to `out/codex-session-auto-rename.mp4`.
The poster renders to `out/poster.png`.

## Storyboard

The video is intentionally brief:

1. A cmux-like workspace opens with opaque UUID and generic session titles.
2. The user notices the noisy sidebar.
3. A small terminal command installs the plugin.
4. A new Codex workspace opens with a clear task title.
5. A fast title wall shows many sessions becoming readable.

The video avoids claiming that the current developer-preview build has solved
every live visible rename path. It presents the product promise while the README
continues to document the current Codex integration limitation.
